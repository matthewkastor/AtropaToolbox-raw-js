<static> atropa.window.open(url, callback, testFn)
Opens a new window and fires a callback once the window has loaded. Optionally, a test function may be provided to fire the callback.
Author: Matthew Christopher Kastor-Inare III
☭ Hial Atropa!! ☭.
Parameters:
{String} url
The url to load in the window.
{Function} callback
The callback function to execute.
{Function} testFn
An optional alternate test for windows containing too many slow loading external resources. If the content you're trying to manipulate is present in the page then you may as well get right to work on it. Return something truthy and the callback will fire, return something falsy and continue waiting. Your test will be tried every 250ms until it returns something truthy.
