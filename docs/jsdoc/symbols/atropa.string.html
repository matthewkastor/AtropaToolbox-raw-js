<static> {Array} atropa.string.escapeCdata(text)
Escapes CDATA sections in text so that the text may be embedded into a CDATA section. This should be run on any text which may contain the string ]]> since said string will effectively end the CDATA section prematurely.
Author: Matthew Christopher Kastor-Inare III
☭ Hial Atropa!! ☭.
Parameters:
{String} text
The text containing CDATA sections to escape.
Returns:
{Array} Returns a string with escaped CDATA sections.
