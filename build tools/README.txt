
Plato is a node module for code analysis https://npmjs.org/package/plato
if you've got node installed you can install plato with npm install -g plato

jsdoc-toolkit is a code documentation tool https://code.google.com/p/jsdoc-toolkit/
jsdoc-toolkit-vsdoc adds a template for generating vsdoc files for intellisense https://code.google.com/p/jsdoc-toolkit-vsdoc/
both are released under the MIT licence http://www.opensource.org/licenses/mit-license.php
